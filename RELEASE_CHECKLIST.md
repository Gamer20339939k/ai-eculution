# Release Checkliste

## Technik
- [ ] `versionCode` erhöht
- [ ] `versionName` gesetzt
- [ ] App startet ohne Crash auf Android 10/12/14
- [ ] Dateiliste lädt, Suche funktioniert, Löschen funktioniert
- [ ] Debug + Release APK wurden in CI gebaut

## Qualität
- [ ] Python-Dateien laufen durch `py_compile`
- [ ] Keine sensiblen Daten im Repo
- [ ] UI-Texte geprüft

## Store Vorbereitung
- [ ] App-Icon final
- [ ] 5+ Screenshots
- [ ] Kurze und lange Beschreibung
- [ ] Datenschutztext vorbereitet
